# ☁ Pass
This is my graduation project. It's a platform for junior software engineers to book mock interviews with Seniors and Techleads. 

### 👨‍💻Technologies used:
- Nextjs
- Typescript
- Zustand for state management
- Sass
- Sass modules
- Material UI date picker
- React-select
- react-paypal-js
- react-toastify
- axios
- Font awesome
- Day js
- Animate on scroll
